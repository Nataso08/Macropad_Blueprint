# Macropad_Blueprint
My personal macropad for Hack Club Blueprint

It contains a OLED display 0.91", a rotay encoder and a 3x4 keyboard matrix. The MCU is a RP2040 XIAO.

It's my first PCB & CAD project ever, and I liked do this. 

## Links:

- [Dashboard](https://blueprint.hackclub.com/home)
- [Tutorial (text)](https://blueprint.hackclub.com/hackpad)

## BOM
- 1x Seeed Studio XIAO RP2040
- 1x 0.91 inch OLED displays
- 1x EC11 Rotary encoder
- 12x MX-Style switches
- 12x 1N4148 Diodes
- 8x M3x5mx4mm heatset inserts
- 8x M3 Screws
